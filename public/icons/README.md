# 图标资源说明

请将以下图标文件放置在此目录：

- `icon-16.png` - 16x16 像素，用于工具栏图标
- `icon-48.png` - 48x48 像素，用于扩展管理页面
- `icon-128.png` - 128x128 像素，用于 Chrome Web Store

## 图标设计要求

- 简洁明了，体现 Cookie/数据同步主题
- 建议使用蓝色或绿色主色调
- 可以使用 Cookie 🍪 或同步 🔄 的图标元素

## 临时方案

开发阶段可以使用在线图标生成工具快速生成占位图标：
- https://favicon.io/
- https://www.favicon-generator.org/
